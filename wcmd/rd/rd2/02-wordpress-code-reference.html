WordPress the_date() Function

the_date() function displays or retrieves the date the current post was written(once per date). So it will ouput the date if the current post's date is different from the previous output.

the_date() Format Parameters

Parameters: All are optional.

  1. $before (string)
    • Text displays before date.
    • Default is ' '
  2. $after (string)
    • Text displays after date.
    • Default is ' '
  3. $format (string)
    • Defines the format of the date.
    • Default is 'Fj,Y' example: "August 26,2024"
    • This uses PHP to format the strings.
  4. $echo (boolean)
    • Wether to echo or return the date.
    • Default is true, which echos the date.
