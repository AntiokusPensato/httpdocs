WordPress History

WordPress History

Wordpress was started because the blogging software b2/cafelog was discontinued. In 2003 Matt Mullenweg and Mike Little built off that software to create WordPress. At first it was just for blogging. Now WordPress powers around 45% of all websites on the web today. In 2004 it changed significantly by adding plugins and themes. By 2011 WordPress was the most popular CMS on the internet.

WordPress.org Entity

WordPress.org is a self-hosted platform. You need to download and install the software on your own server or hosted server. This gives absolute control allowing highly customizable content. It is also an open-source projects maintained by thousands of people worldwide.

WordPress.com Entity

WordPress.com is a hosted platform. Considered freemium. The platform is owned by Automattic which was founded by Matt Mullenweg. Being it is hosted users don't need to think about hosting, security, backups, or updates. You do not need any sort of technical knowledge to use WordPress.com

Summary

While both sites run the same software the main difference between the two is WordPress.org is self-hosted and WordPress.com is hosted. In 2010 Automattic transfered the trademark ownership to the WorPress Foundation, a non-profit organization.

WordPress Foundation's Mission Statement:

"The point of the foundation is to ensure free access, in perpetuity, to the software projects we support. People and businesses may come and go, so it is important to ensure that the source code for these projects will survive beyond the current contributor base, that we may create a stable platform for web publishing for generations to come."
