WordPress Code Author Functions

The the_author() Function

We use this function to display the authors name of the current post. This just shows who wrote each post.

The the_author_link() Function

This function is similar to the_author() but gives it a little somthing extra. Now it will display the author's name as a clickable link the brings you directly to that author's archive page. From there you can see all posts by said author.

Differences and Similarities

Both of these functions display the authors name. Differences between the two would be the_author() only outputs the author's name as plain text. While the_author_link() wraps the author's name in an anchor tag which makes it a clickable link.

Similarities between the two would be they both rely on the current post in the WordPress loop to determine which author to display and neither function requires any parameters to work.

Summary

In wordpress you can use both the_author() and the_author_link() to display author info. You should use one or the other to give attribution to content and to its creators. If you don't want or need to provide access to all of an author's posts use the_author() if you do need or want to use the_author_link().
